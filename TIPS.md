# CLI tips

This is for tips when using the cli

## Tips for using the wrong cli arg

- `uninstall`
  - use `remove` instead

- `install`
  - use `add` instead

- `compile`
  - use `build` instead

- `execute`
  - use `run` instead

- `create`
  - use `new` instead

- `package`
  - use `bundle` instead
